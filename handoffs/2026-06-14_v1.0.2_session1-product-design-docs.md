## Architecture Decision
**No database.** Gemini Council (3 REVISE votes) caught that Neon + Drizzle was over-engineered for a 30-plan dataset. Final design: weekly AA snapshots committed as `src/data/aa-snapshots/YYYY-MM-DD.json`, computed scores regenerated at each build to `src/data/computed-scores.json`. No runtime DB queries, no cold starts. Git history is the version history.
