### Dev server note

`pnpm dev` (Turbopack) throws `localStorage.getItem is not a function` under Node.js v25. Root cause: Node v25 adds a built-in `localStorage` stub that requires `--localstorage-file` to function. This is NOT introduced by this session's changes — confirmed by the production build (`pnpm build` + `pnpm start`) working cleanly with the pages serving correctly.
